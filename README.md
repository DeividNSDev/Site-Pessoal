# 💻 Meu Portfólio Pessoal

Este é o repositório do meu site pessoal, desenvolvido para centralizar minha trajetória acadêmica, habilidades técnicas e projetos. Como estudante de Sistemas de Informação na UNIFEBE, utilizo este espaço como meu cartão de visitas no ecossistema de tecnologia.

---

## 🚀 Tecnologias e Ferramentas

O projeto foi construído focando nos fundamentos da Web para garantir um código limpo e performático:

* **HTML5**: Estruturação semântica de conteúdo.
* **CSS3**: Estilização moderna utilizando variáveis (Custom Properties) e Flexbox para layout responsivo.
* **Git & GitHub**: Controle de versão e gerenciamento de repositório remoto.

---

## 🎨 Design e UI/UX

A escolha visual seguiu uma linha **Clean e Profissional**, utilizando uma paleta de cores baseada em:
* **Cinza e Azul**: Transmite confiança, seriedade e tecnologia.
* **Contraste Leve**: Fundo claro para priorizar a legibilidade e o conforto visual durante a leitura.
* **Componentes**: Uso de *cards* para formação acadêmica e *tags* interativas para habilidades.
